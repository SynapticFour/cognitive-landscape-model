# Changelog

All notable changes to this project are documented here. The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

### Added

- Five-layer research monorepo: `@clm/core`, `@clm/landscape`, `@clm/dynamics`, `@clm/structural`, `@clm/trajectory-explanation`, `@clm/interpretation`.
- Vector-based `CognitiveState` / `StateSpace` API with PCMS preset and trajectory serialization.
- `TransitionEngine`, `SimulationRunner`, and structural modulation via `ProbabilisticDynamicsModulator`.
- Latent `StructuralModel`, `CalibrationEngine`, abstract and PCMS structural presets.
- Trajectory explanation engine (mechanistic causal narratives) and interpretation plugins (non-causal worldviews).
- Integration, boundary, and golden-vector test suites.
